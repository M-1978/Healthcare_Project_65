# Healthcare Data Analysis System using Python

## 📌 Problem Statement
Analyze healthcare dataset to extract meaningful insights.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## ⚙️ How to Run
1. Install Python
2. Install libraries:
   pip install pandas numpy matplotlib seaborn
3. Run the code

## 📊 Output
- Gender distribution chart
- Age histogram
- Medical condition analysis
- Correlation heatmap

## 🎯 Conclusion
This project helps in understanding healthcare data trends and improves decision-making.
